Pet Shop Website : 
   - A simple Pet Shop Website built using HTML and CSS.

About the Project : 
    This is a beginner-level frontend project where I converted my UI design into a real website.  
Users can explore different pets like dogs, cats, and birds through a card-based interface.
The main goal of this project is to improve my UI/UX and basic web development skills

Features
  - Pet listing with images  
  - Clean UI design  
  - Navigation bar  
  - Basic responsive layout  

Tech Used
  - HTML  
  - CSS  

Run
   Open `index.html` in your browser.
